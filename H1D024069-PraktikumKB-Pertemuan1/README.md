# Pertemuan 1 — Dasar Pemrograman Python

Implementasi dasar pemrograman Python menggunakan **struktur data**, **struktur kontrol**, dan **library bawaan Python**.

---

## Source Code

### SimulasiTokoKopi.py
Program simulasi pemesanan kopi berbasis CLI yang mendemonstrasikan penggunaan berbagai konsep dasar Python.

---

## Konsep yang Diimplementasikan

### Struktur Data
| Struktur | Penggunaan |
|----------|------------|
| list | Menyimpan daftar menu kopi dan histori pesanan |
| dict | Menyimpan resep kopi (level, harga, waktu seduh) |
| set | Mencatat jenis kopi unik yang pernah dipesan |

### Struktur Kontrol
| Struktur | Penggunaan |
|----------|------------|
| for | Menampilkan menu dan histori pesanan |
| while | Perulangan menu utama |
| if-elif-else | Validasi pilihan menu |
| try-except | Validasi input angka |

### Library Python
| Library | Penggunaan |
|---------|------------|
| random | Rekomendasi kopi acak & variasi suhu |
| time | Efek animasi brewing dan waktu pesanan |
| math | Perhitungan harga setelah diskon |

---

## Fitur Program

- **Pesan Kopi** — memilih kopi dari menu, dilengkapi efek brewing dan suhu acak
- **Random Kopi** — rekomendasi kopi acak dengan harga diskon 10%
- **Histori Pesanan** — melihat seluruh riwayat pesanan beserta waktu
- **Statistik Unik** — menampilkan variasi kopi yang sudah & belum dipesan

---

## Cara Menjalankan

```bash
python SimulasiTokoKopi.py
```
